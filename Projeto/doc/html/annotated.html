Estruturas de Dados
Aqui estão as estruturas de dados, uniões e suas respectivas descrições:
 CArvoreContem as variáveis necessárias para criar um no do tipo proprietário em uma arvore binária
 CDataContem informacoes sobre uma data
 CEnderecoGuarda um endereco
 CManutencaoContem informacoes sobre uma manutencão
 COficinaContem os dados da oficina
 CProprietarioGuarda as informacoes de um proprietário
 CTelefoneGuarda um telefone
 CVeiculoContem as informacoes de um veiculo
